## What is JustGo?
저스트고는 무작위 장소 추천 서비스입니다.  
내 현재 위치, 소요할 수 있는 시간, 가고싶은 곳의 주제(백화점, 박물관 등)를 선택하면 그에 맞게 관광지를 추천해주고, 관광지에 도착하기 전까지 목적지를 알려주지 않고 네비게이션을 따라 가는 서비스입니다.

## 사용 기술
Kotlin, MVVM, ConstraintLayout, etc..
